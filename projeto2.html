Análise de Dados em Saúde Pública

Faculdade de Medicina
Saúde
3 meses
Intermediário

Projeto de análise de dados de saúde pública utilizando técnicas estatísticas e de visualização para identificar padrões, tendências e fatores de risco em populações específicas. Foco em doenças crônicas e epidemiológicas.

Descrição do Projeto

Este projeto tem como objetivo analisar dados de saúde pública para identificar padrões e tendências em doenças crônicas e epidemiológicas. Utilizando técnicas de mineração de dados e visualização, os participantes irão trabalhar com conjuntos de dados reais do SUS e outros sistemas de saúde.

O projeto é ideal para estudantes de medicina, saúde pública, estatística e ciência de dados que desejam aplicar seus conhecimentos em problemas reais de saúde coletiva.

Objetivos

  • Identificar padrões de incidência de doenças em diferentes regiões
  • Analisar fatores socioeconômicos relacionados à saúde pública
  • Desenvolver visualizações claras e informativas para tomada de decisão
  • Criar modelos preditivos para surtos de doenças sazonais
  • Propor intervenções baseadas em evidências para políticas públicas

Materiais Utilizados

  • Python com bibliotecas de análise de dados (Pandas, NumPy, SciPy)
  • R para análise estatística avançada
  • Tableau/Power BI para visualização de dados
  • Bancos de dados do DATASUS e IBGE
  • Jupyter Notebooks para documentação e análise
  • GitHub para controle de versão e colaboração

Etapas de Desenvolvimento

  1. Coleta e limpeza de dados de saúde pública (2 semanas)
  2. Análise exploratória e estatística descritiva (3 semanas)
  3. Desenvolvimento de visualizações e dashboards (3 semanas)
  4. Modelagem estatística e preditiva (4 semanas)
  5. Interpretação dos resultados e elaboração de relatório (2 semanas)
  6. Apresentação dos resultados para stakeholders (1 semana)
