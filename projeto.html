App de Reciclagem Sustentável

Universidade Federal de Tecnologia
Tecnologia
2 meses
Iniciante

Desenvolvimento de um aplicativo mobile que ajuda usuários a identificar tipos de materiais recicláveis e localizar pontos de coleta próximos. O projeto inclui interface amigável e sistema de recompensas.

Descrição do Projeto

O aplicativo de Reciclagem Sustentável tem como objetivo facilitar o processo de reciclagem para a população em geral, incentivando práticas sustentáveis através da tecnologia. Com uma interface intuitiva, os usuários podem escanear materiais para identificar se são recicláveis, encontrar pontos de coleta próximos e acompanhar seu impacto ambiental.

O projeto é ideal para iniciantes em tecnologia que desejam aprender sobre desenvolvimento mobile, design de interface e trabalho com APIs de geolocalização.

Objetivos

  • Conscientizar sobre a importância da reciclagem
  • Facilitar o descarte correto de materiais recicláveis
  • Criar uma rede de pontos de coleta colaborativos
  • Implementar sistema de gamificação para incentivar usuários
  • Fornecer dados sobre o impacto ambiental gerado pela reciclagem

Materiais Utilizados

  • React Native para desenvolvimento do app
  • Firebase para backend e autenticação
  • Google Maps API para geolocalização
  • Figma para prototipagem da interface
  • TensorFlow Lite para reconhecimento de imagens
  • GitHub para controle de versão

Etapas de Desenvolvimento

  1. Pesquisa e definição de requisitos (1 semana)
  2. Prototipação da interface do usuário (2 semanas)
  3. Desenvolvimento do frontend (3 semanas)
  4. Implementação do backend (2 semanas)
  5. Integração com APIs de geolocalização (1 semana)
  6. Testes com usuários (2 semanas)
  7. Publicação na Play Store e App Store (1 semana)
